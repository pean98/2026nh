# 인천논현고등학교 2026 교육과정박람회 웹사이트 통합 기획안 v2

## 1. 기획 콘셉트

인천논현고 교육과정박람회 웹사이트는 PDF 가이드북을 온라인으로 옮겨 놓는 사이트가 아니라, 학생이 자신의 진로와 과목 선택을 직접 탐색하고 설계해 보는 디지털 교육과정 박람회로 기획한다.

핵심 문장:

> 내 꿈을 향한 주도적인 선택, 인천논현고 교육과정박람회

## 2. 핵심 목적

### 정보의 접근성 혁신

60쪽 안팎의 1학년·2학년 과목 선택 워크북을 모바일과 웹에서 쉽게 탐색할 수 있도록 재구성한다. 학생은 PDF 전체를 넘기지 않아도 학년, 교과군, 선택 유형, 키워드, 진로 계열로 필요한 과목을 찾을 수 있다.

### 자기주도적 교육과정 설계 지원

고교학점제의 주요 기준을 학생이 직접 점검할 수 있도록 한다.

- 3년간 총 192학점 이수
- 교과 174학점 + 창의적 체험활동 18학점
- 필수 이수 학점 84학점 이상
- 국어·수학·영어 교과군 81학점 초과 불가
- 과목별 위계와 선수 과목 관계 확인

### 진로와 대입 연계 시각화

학생의 희망 계열, 학과, 대학에 따라 권장 선택과목을 보여주고, 학교 선택 과목과 어떻게 연결되는지 시각화한다. 단, 대학별 권장과목은 매년 달라질 수 있으므로 “상담 참고 자료” 성격을 명확히 표시한다.

### 상담 효율 향상

학생이 관심 과목, 선택 이유, 고민 과목, 질문을 미리 정리한 뒤 담임·교과·진로 교사와 상담할 수 있도록 돕는다.

## 3. 주요 사용자

### 학생

- 학년별 선택 과목을 빠르게 확인하고 싶은 학생
- 진로는 있지만 어떤 과목을 들어야 할지 모르는 학생
- 관심 과목을 담아 보고 학점 기준을 점검하고 싶은 학생
- 상담 전에 질문을 정리하고 싶은 학생

### 학부모

- 고교학점제와 2022 개정 교육과정을 이해하고 싶은 학부모
- 자녀의 과목 선택이 대입과 어떻게 연결되는지 알고 싶은 학부모
- 과목 선택 상담 전에 기본 정보를 확인하고 싶은 학부모

### 교사

- 학생의 과목 선택 후보와 고민 지점을 빠르게 파악하고 싶은 교사
- 상담 전 학생의 선택 계획을 확인하고 피드백하고 싶은 교사
- 박람회 자료를 반복 설명하지 않고 학생별 맞춤 상담에 집중하고 싶은 교사

## 4. 정보 구조

### 1. 홈: Main Dashboard

홈은 홍보성 랜딩 페이지가 아니라 바로 탐색할 수 있는 대시보드형 첫 화면으로 구성한다.

주요 구성:

- 타이틀: “인천논현고 2026 교육과정박람회”
- 서브 문구: “내 꿈을 향한 주도적인 선택”
- 학년 선택: 1학년 가이드, 2학년 가이드
- 빠른 메뉴: 선택 과목 백과, 학점 계산기, 진로·대입 매칭, 꿈두레 공동교육과정
- 핵심 체크 카드: 192학점, 교과 174학점, 필수 84학점, 국수영 81학점 제한
- 검색창: 과목명, 키워드, 계열, 학과 검색

### 2. 고교학점제 가이드

워크북의 공통 안내 내용을 짧고 시각적인 정보 단위로 제공한다.

하위 콘텐츠:

- 고교학점제란 무엇인가
- 2022 개정 교육과정의 과목 구조
- 공통 과목/일반 선택/진로 선택/융합 선택 비교
- 성적 산출 방식: 성취도, 석차등급, P 이수 여부
- 5등급제 누적 비율 인포그래픽
- 졸업 이수 기준
- 과목 선택 4단계: 기준 확인 → 진로 탐색 → 과목 선택 및 상담 → 학업 계획 수립
- 대학 입시와 과목 선택의 관계

### 3. 선택 과목 백과

사이트의 중심 콘텐츠다. 1학년 26개 과목, 2학년 34개 과목을 디지털 카드와 상세 페이지로 제공한다.

필터:

- 학년: 1학년, 2학년
- 교과군: 국어, 수학, 영어, 사회, 과학, 예술, 정보, 제2외국어/한문, 교양
- 선택 유형: 일반 선택, 진로 선택, 융합 선택
- 평가 방식: 석차등급, 성취도, P
- 수능 출제 여부
- 관심 계열: 인문, 사회, 상경, 교육, 자연, 공학, 의약학, 예술·체육, 국제
- 키워드: 데이터분석, 비판적사고, 스토리텔링, 환경, 심리, 의사소통 등

과목 카드:

- 과목명
- 학년
- 교과군
- 선택 유형
- 편성 학점
- 평가 정보
- 수능 출제 여부
- 한 줄 소개
- 대표 키워드
- 후보 담기 버튼

과목 상세:

- 과목 프로필
- 과목을 여는 한 줄
- 주요 키워드
- 무엇을 배울 수 있나요?
- 대영역별 핵심 질문과 학습 활동

- 함께 보면 좋은 과목
- 상담 질문 작성

### 4. 진로 및 대입 매칭

과목명에서 출발하지 못하는 학생을 위해 진로와 학과 중심의 탐색을 제공한다.

기능:

- 계열별 추천 과목 보기
- 학과 키워드 검색
- 대학별 권장과목 조회
- 내 관심 과목과 권장과목 비교
- 부족한 과목 또는 보완 과목 표시

주의 문구:

- 대학별 권장과목은 매년 변동될 수 있음
- 최종 지원 전 반드시 대학 입학처 안내 확인 필요
- 본 기능은 과목 선택 상담을 돕는 참고 자료임

### 5. 나만의 교육과정 시뮬레이터

웹사이트의 핵심 차별화 기능이다. 학생이 장바구니에 과목을 담듯 2·3학년 선택 과목을 조합해 보고, 고교학점제 기준을 점검할 수 있게 한다.

주요 기능:

- 관심 과목 담기
- 학년별/학기별 과목 배치
- 교과군별 학점 합계
- 총 교과 학점 계산
- 국어·수학·영어 교과군 합계 계산
- 필수 이수 학점 충족 여부 표시
- 과목 유형 비율 확인
- 진로 계열과의 연결성 표시
- 위계 과목 경고
- 상담 메모 작성
- 결과 저장 및 PDF 출력

경고 예시:

- “국어·수학·영어 교과군 합계가 81학점을 초과했습니다.”
- “과학 교과 필수 이수 학점이 부족합니다.”
- “미적분Ⅱ 선택 전 미적분Ⅰ 이수 여부를 확인하세요.”
- “선택 과목이 특정 교과군에 집중되어 있습니다. 진로와 이수 기준을 함께 상담해 보세요.”

### 6. 꿈두레 공동교육과정

학교 맞춤형 실용 정보로 별도 메뉴를 둔다.

내용:

- 꿈두레 공동교육과정이란?
- 거점형, 밴드형, 온라인형 차이
- 인천논현고 미개설 과목을 수강할 수 있는 방법
- 신청 시기와 유의사항
- 학교 상담 필요 항목
- 관련 링크 또는 안내 자료

## 6. 핵심 사용자 흐름

### 학생 흐름

1. 홈에서 학년 선택
2. 관심 키워드 또는 계열 검색
3. 선택 과목 백과에서 과목 탐색
4. 과목 상세에서 학습 내용 확인
5. 관심 과목을 시뮬레이터에 담기
6. 학점과 이수 기준 점검
7. 진로·대입 매칭에서 권장과목 비교
8. 상담 질문 작성
9. 학업 계획서 출력 또는 저장

### 학부모 흐름

1. 고교학점제 가이드 확인
2. 성적 산출 방식과 졸업 기준 이해
3. 자녀가 담은 과목 목록 확인

5. 상담 준비실에서 질문 함께 정리

## 7. UI/UX 원칙

### 모바일 우선

학생은 교실, 집, 박람회 현장에서 스마트폰으로 접속할 가능성이 높다. 모바일에서 과목 검색, 필터, 후보 담기, 상담 질문 작성이 모두 가능해야 한다.

### 태그 중심 탐색

워크북의 주요 키워드를 해시태그로 전환한다.

예시:

- #비판적사고
- #데이터분석
- #스토리텔링
- #환경감수성
- #의사소통
- #세계시민
- #문제해결
- #심리
- #AI
- #지속가능성

태그를 누르면 관련 과목이 모아져야 한다.

### 과목 비교는 단순하게

학생이 고민하는 과목 2~3개를 나란히 비교할 수 있게 한다.

비교 항목:

- 교과군
- 선택 유형
- 학점
- 평가 방식
- 수능 출제 여부
- 주요 키워드
- 추천 학생
- 관련 계열

### 경고는 무섭지 않게

시뮬레이터의 경고 문구는 학생을 막는 느낌이 아니라 상담을 유도하는 톤으로 작성한다.

예:

- “기준을 다시 확인해 볼 필요가 있어요.”
- “이 조합은 담임 선생님과 상담해 보세요.”
- “관심 진로와 연결되는 보완 과목을 함께 살펴볼까요?”

## 9. 개발 우선순위

### 1차 MVP

박람회 당일 바로 쓸 수 있는 탐색형 웹사이트.

- 홈 대시보드
- 고교학점제 가이드
- 선택 과목 백과
- 과목 검색/필터
- 과목 상세 페이지
- 태그 검색
- PDF 자료실

### 2차

학생 참여형 기능 추가.

- 관심 과목 담기
- 과목 비교
- 나만의 교육과정 시뮬레이터 기본형
- 국수영 81학점 제한 경고
- 상담 질문 작성
- 학업 계획서 PDF 출력

### 3차

진로·대입 연계 강화.

- 계열별 추천 과목
- 대학별 권장과목 검색
- 내 선택 과목과 권장과목 매칭
- 꿈두레 공동교육과정 안내
- 교사용 상담 보기

## 10. 제작 시 유의사항

- 대학별 권장과목은 최신성이 중요하므로 관리자 수정이 쉬운 구조로 만든다.
- 권장과목 매칭은 “합격 가능성 판단”처럼 보이지 않게 표현한다.
- 학점 시뮬레이터는 실제 수강 신청 시스템이 아니라 상담 보조 도구임을 명시한다.
- PDF 원문은 자료실에 유지하되, 학생이 실제로 보는 화면은 카드와 상세 페이지 중심으로 구성한다.
- 과목 설명은 길게 펼쳐 두지 않고, 필요한 만큼 단계적으로 열어 보게 한다.
- 모바일에서 검색창, 필터, 후보 담기 버튼이 항상 쉽게 접근 가능해야 한다.

## 11. 최종 제안

이 사이트는 다음 순서로 구현하는 것이 가장 현실적이다.

1. 먼저 모든 과목을 카드형 데이터로 정리한다.
2. 학년/교과군/선택 유형/키워드 필터를 만든다.
3. 과목 상세 페이지를 만든다.
4. 관심 과목 담기와 간단한 비교 기능을 붙인다.
5. 학점 시뮬레이터를 추가한다.
6. 진로·대입 매칭과 꿈두레 공동교육과정 안내를 확장한다.

첫 화면은 안내문보다 “내 학년 선택”, “과목 검색”, “학점 계산기”가 먼저 보여야 한다. 학생에게 필요한 것은 설명을 읽는 경험보다 선택을 시작하는 경험이다.
