# 인천논현고등학교 교육과정 박람회 웹사이트 기획안

## 1. 기획 방향

2026학년도 1학년/2학년 과목 선택 워크북을 단순 PDF 열람 자료가 아니라, 학생이 자신의 진로와 흥미에 맞춰 과목을 탐색하고 선택 계획을 세울 수 있는 웹 기반 박람회로 전환한다.

핵심 목표는 다음 세 가지다.

- 학생: 과목을 쉽고 빠르게 비교하고, 나에게 맞는 선택 과목을 찾는다.
- 학부모: 고교학점제, 평가 방식, 대입 연계 정보를 한눈에 이해한다.
- 교사: 상담 전 학생의 관심 과목과 질문을 미리 파악해 상담 효율을 높인다.

## 2. 자료 기반 핵심 콘텐츠

### 공통 안내 콘텐츠

두 학년 자료에 공통으로 포함된 내용이다.

- 2022 개정 교육과정과 고교학점제
- 과목 분류: 공통 과목, 일반 선택, 진로 선택, 융합 선택
- 교과군별 과목 체계
- 과목별 성적 산출 방식: 원점수, 성취도, 석차등급, 성취도별 분포비율, 과목평균, 수강자수
- 3년간 총 이수 학점: 교과 174학점 + 창의적 체험활동 18학점 = 총 192학점
- 필수 이수 학점: 총 84학점 이상
- 국어·수학·영어 교과군 이수 학점 총합 81학점 초과 불가
- 과목 선택 절차: 이수 기준 확인 → 진로/학과 정보 탐색 → 과목 선택 및 상담 → 학업 계획 수립
- 대입 전형과 과목 선택의 관계
- 학생부종합전형 평가 요소: 학업역량, 진로역량, 공동체역량
- 계열별 권장 선택 과목 안내

### 1학년 선택 과목 콘텐츠

1학년 워크북의 선택 과목 안내는 총 26개 과목 중심으로 구성한다.

- 국어: 독서 토론과 글쓰기, 문학과 영상
- 수학: 기하, 경제 수학
- 영어: 미디어 영어, 세계 문화와 영어
- 사회: 세계시민과 지리, 도시의 미래 탐구, 세계사, 동아시아 역사 기행, 법과 사회, 정치, 윤리와 사상, 현대사회와 윤리
- 과학: 물리학, 역학과 에너지, 화학, 물질과 에너지, 생명과학, 세포와 물질대사, 지구과학, 지구시스템과학
- 정보: 정보, 데이터 과학
- 제2외국어/한문: 제2외국어(중국어/일본어), 제2외국어 회화(중국어 회화/일본어 회화)

### 2학년 선택 과목 콘텐츠

2학년 워크북의 선택 과목 안내는 총 34개 과목 중심으로 구성한다.

- 국어: 매체 의사소통, 언어생활 탐구
- 수학: 미적분Ⅱ, 실용 통계
- 영어: 심화 영어, 영미 문학 읽기
- 사회: 사회와 문화, 경제, 사회문제 탐구, 한국지리 탐구, 기후변화와 지속가능한 세계, 인문학과 윤리, 윤리문제 탐구, 역사로 탐구하는 현대 세계
- 과학: 전자기와 양자, 화학 반응의 세계, 생물의 유전, 행성우주과학, 과학의 역사와 문화, 융합과학 탐구, 기후변화와 환경 생태
- 예술: 음악 연주와 창작, 음악과 미디어, 미술 창작, 미술과 매체
- 정보: 인공지능 기초, 소프트웨어와 생활
- 제2외국어/한문: 중국문화, 심화 중국어, 일본문화, 심화일본어
- 교양: 생태와 환경, 인간과 심리, 논술

## 3. 사이트 정보 구조

### 1. 홈

첫 화면은 박람회 안내보다 바로 탐색으로 들어가게 설계한다.

- 상단: “2026 인천논현고 교육과정 박람회”
- 빠른 진입 버튼: 1학년 과목 보기, 2학년 과목 보기, 진로별 추천 보기, 나의 과목 설계하기
- 핵심 알림: 총 이수 학점, 필수 이수 학점, 국수영 81학점 제한, 상담 전 확인 사항

### 2. 고교학점제 한눈에 보기

워크북의 공통 설명을 짧은 카드와 인포그래픽으로 재구성한다.

- 고교학점제란?
- 과목 유형 비교: 공통/일반/진로/융합
- 성적 산출 방식 비교
- 졸업 이수 기준
- 과목 선택 4단계

### 3. 과목 탐색

사이트의 핵심 화면이다.

필터:

- 학년: 1학년, 2학년
- 교과군: 국어, 수학, 영어, 사회, 과학, 예술, 정보, 제2외국어/한문, 교양
- 선택 유형: 일반 선택, 진로 선택, 융합 선택
- 평가 방식: 석차등급 있음/없음, 성취도 5단계, P 처리
- 수능 출제 여부
- 관심 계열: 인문, 사회, 경영, 자연, 공학, 의학·보건, 교육, 예술·체육, 국제 등

과목 카드:

- 과목명
- 교과군/선택 유형
- 편성 학점
- 평가 정보
- 수능 출제 여부
- 한 줄 소개
- 주요 키워드
- 추천 학생

과목 상세:

- 과목 프로필
- “무엇을 배울 수 있나요?”
- 대영역별 핵심 질문과 주요 학습 활동
- 선배가 들려주는 심화 학습 활동
- 이런 학생에게 추천해요
- 관련 계열/진로
- 함께 보면 좋은 과목
- 상담 질문 작성 버튼

### 4. 진로별 과목 찾기

학생이 과목명보다 진로에서 출발할 수 있게 한다.

- 관심 계열 선택
- 계열별 권장 과목 경향 안내
- 해당 계열과 연결되는 과목 묶음 표시
- “필수”가 아니라 “권장 경향”임을 명확히 표시
- 대학별 세부 권장 과목은 반드시 대학 안내를 확인하도록 안내

### 5. 나의 과목 설계

학생이 선택 후보를 담고 상담 전 점검할 수 있는 기능이다.

- 관심 과목 담기
- 학점 합계 보기
- 국어·수학·영어 81학점 제한 경고
- 교과군별 필수 이수 기준 체크
- 일반/진로/융합 선택 비율 확인
- 관심 계열과의 연결성 메모
- 상담 질문 작성
- PDF 또는 이미지로 저장

### 6. 상담 준비실

박람회 현장 또는 온라인 상담 전에 학생이 질문을 정리하도록 돕는다.

- 내가 담은 과목 목록
- 왜 이 과목에 관심 있는지
- 희망 진로/학과
- 고민되는 과목 비교
- 담임/교과/진로 교사에게 물어볼 질문

### 7. 자료실

- 1학년 워크북 PDF 다운로드
- 2학년 워크북 PDF 다운로드
- 학업 계획서 양식
- 계열별 권장 과목 참고 자료
- 학교 홈페이지 링크

## 5. 화면 설계 초안

### 홈 화면

- 좌측 또는 상단: 학년 탭
- 중앙: 과목 검색창
- 하단: “고교학점제 핵심 체크” 4개 요약 카드
- 모바일에서는 검색창과 학년 탭을 최상단 고정

### 과목 탐색 화면

- 데스크톱: 좌측 필터, 우측 과목 카드 그리드
- 모바일: 필터 바텀시트, 과목 카드 리스트
- 검색 결과 상단에 “총 N개 과목”과 적용 필터 표시
- 카드 색상은 교과군별로 구분하되 과하지 않게 사용

### 과목 상세 화면

- 상단: 과목명, 교과군, 선택 유형, 학점, 평가, 수능 여부
- 본문 탭: 소개 / 배움 내용 / 심화 활동 / 추천 학생 / 상담 메모
- 하단 고정 버튼: 후보 담기, 비교하기

### 나의 과목 설계 화면

- 선택 후보 목록
- 학점 합계와 제한 조건 체크
- 교과군별 이수 현황
- 진로 연계 메모
- 출력/저장 버튼

## 7. 데이터 구조 제안

과목 데이터는 정적 JSON 또는 스프레드시트 기반 CMS로 관리한다.

필수 필드:

- id
- grade
- subjectName
- subjectGroup
- selectionType
- credits
- evaluationType
- csatIncluded
- oneLine
- keywords
- learningAreas
- advancedActivities
- recommendedFor
- relatedMajors
- sourceWorkbook

예시:

```json
{
  "id": "g2-human-psychology",
  "grade": 2,
  "subjectName": "인간과 심리",
  "subjectGroup": "교양",
  "selectionType": "진로 선택",
  "credits": "2~4학점",
  "evaluationType": "P",
  "csatIncluded": false,
  "oneLine": "과학적 증거로 마음의 원리를 이해하고, 타인과 공존하며 나다운 행복을 찾아가는 과목입니다.",
  "keywords": ["과학적탐구", "뇌", "자기효능감", "회복탄력성", "행복"]
}
```

## 8. 디자인 톤

- 대상: 고등학생, 학부모, 교사
- 분위기: 신뢰감 있는 학교 공공 서비스 + 학생 친화적 탐색 도구
- 색상: 교과군별 포인트 컬러를 쓰되 전체는 흰색/밝은 회색 중심
- 타이포그래피: 긴 설명은 카드보다 본문형 레이아웃으로 가독성 확보
- 아이콘: 학점, 평가, 수능 여부, 추천 계열 등 반복 정보에 아이콘 사용
- 접근성: 모바일 우선, 검색/필터 버튼 충분한 터치 영역, PDF 없이도 핵심 내용 열람 가능

## 9. 운영 및 제작 메모

- 원본 PDF는 자료실에 보관하되, 학생이 실제로 쓰는 화면은 과목 카드/상세 페이지로 재구성한다.
- 과목 설명은 워크북 원문을 그대로 길게 노출하기보다 “한 줄 소개 → 핵심 키워드 → 배움 내용 → 추천 학생” 순서로 단계적으로 보여준다.
- 계열별 권장 과목은 확정 추천처럼 보이지 않게 “경향”, “상담 참고” 표현을 사용한다.
- 학점 계산 기능은 최종 수강 신청 시스템이 아니라 상담 보조 도구임을 명시한다.
- 박람회 당일 사용을 고려해 QR 접속, 모바일 검색, PDF 다운로드, 상담 질문 저장 기능을 우선 구현한다.

## 10. 추천 첫 구현 범위

첫 버전은 “과목 탐색형 웹사이트”로 작게 시작하는 것이 좋다.

1. 홈
2. 고교학점제 안내
3. 학년별 과목 탐색
4. 과목 상세
5. 자료실

이후 학생 반응을 보며 관심 과목 담기, 학점 체크, 상담 대시보드를 확장한다.
